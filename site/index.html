                Opening on macOS
              

              
macOS blocks unsigned apps once. After downloading:

              
    
                
  1. Open Captura, then click Done on the warning.
    2. 
                
  2. Go to System Settings → Privacy & Security, scroll to Security, and click Open Anyway.
    3. 
              

              
Or run this once in Terminal:

              xattr -dr com.apple.quarantine /Applications/Captura.app
            

            

              

                
                Opening on Windows
              

              
SmartScreen warns once for unsigned installers:

              
    
                
  1. Run the installer. If you see "Windows protected your PC", click More info.
    2. 
                
  2. Click Run anyway to continue.
    3. 
              

              
These warnings appear because the app is open-source and unsigned — not because anything's wrong. Every download lists a SHA-256 checksum you can verify.
